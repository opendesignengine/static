RE: Accounting software review - Added by J. Simmons over 6 years ago

In looking over software I've identified a few questions we have to answer:

  • How much do we care about manufacturing?
    I would say ideally we want to find something that we can grow into (including manufacturing), but I would not make that a blocker in terms of qualifying tools (just highly desirable)
  • Do we care if it has to have a server?
    I think this is up to Bryan, as he is much more likely to manage any servers for the time being (I obviously can, but my travel and more recent work focus makes me less ideal as an admin)
  • If we're willing to host a server, how would we do that? Where?
    As long as we had offsite backups, I would be fine running it at someone's home. In fact, prefer that to paying for server space (less random person access that way and lower cost)
  • How concerned are we about this being a long term solution?
    I think we are in a place where if we can set our selves up for a long term solution, that is good, but like my comment about manufacturing, I don't think we are in a place where that should be a must have
  • How mature do we want the platform?
    Fairly mature... I don't think we want our records to be inaccessible due to platform maturity issues

Note, Bryan has the expertise in this field, so I will likely defer to his more informed opinion (while of course offering input on values and concerns)
