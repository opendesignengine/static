Great news from Ipython-Land (actually Jupyter-Land)

Added by J. Simmons about 6 years ago

Check this link Jeremy sent me:

http://blog.jupyter.org/2016/07/14/jupyter-lab-alpha/

Look familar? Check out the wiki landing page, because it should look familar.

These upgrades should take a good chunk of work off our plate for MTK. But there are a couple of things I still want to see in MTK that are not discussed in the blog post.

  • I would like to see MTK be an application, managing the server process and giving me a web container for notebooks other than my already busy browser
  • I still want to see the Python libraries we use included in MTK
  • I want to see a focus on easy generation of professional looking PDFs
  • There are also a number of Jupyter plugins we use and those need to be in MTK

Still, packing things up is way better than extending core functionality.
