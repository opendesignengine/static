Open Design Engine

Home

Open Design Engine (ODE) is a web portal developed specifically to address the urgent need to publish not only the final plans, but the entire design process for open source hardware projects.

Open Design Engine provides:

  • Full set of per-project features: Projects have all the key features you expect in a project management tool
    • Issue tracking
    • News
    • Files
    • Wiki
    • Repositories: SVN (Git coming soon)
    • DMSF: Document management system for project files beyond source code, includes versioning and drag-and-drop interface
    • Forums
    • Roadmap
  • First class sub-projects: Projects in ODE may be organized into sub-projects which have all of the features and customization of top-level projects.
  • Consistent Wiki-like syntax: ODE uses its Wiki syntax throughout the site. ODE also includes some additional markups in its Wiki syntax
    • Embed YouTube and Vimeo videos
    • Embed Pledgie, Kickstarter, and Paypal Badges/Buttons
    • Embed issue details

Sign up for an account today and start your own open source hardware project or help out with an existing one.

To learn more about ODE and its features check out the ODE roadmap and the the ODE Project Wiki.
