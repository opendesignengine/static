Switch to 5 Volt Version of MLX90614 IR Sensor

Description

There are both 3.3 volt and 5 volt versions of the MLX non-contact IR sensor we're using. We currently have the 3.3 volt version for Shepard v1.1. We need to switch to the 5 volt version so that it and the load cell can be on a common voltage.
