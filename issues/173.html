Bug #174: Create Java based GUI for Shepard DCS

Add a "wizard" mode for the Shepard GUI

Added by Christopher Sigman about 9 years ago. Updated about 9 years ago.

Status:New Start date:09/12/2013
Priority:Normal Due date:
Assignee:Christopher Sigman % Done:

0%
Category:-
Target version:v2.0 Shepard Test Stand - Prototype

Description

The current GUI is set up in more of an "expert" format, where all of the options for controlling the hardware and GUI are immediately available. It would be good for many situations though to have the control options simplified into a guided, step-by-step format. The steps would be:
  1. Connect
  2. Calibrate
  3. Set Test Notations
  4. Record
  5. Stop Recording
  6. Clear
    After the last step, this test "wizard" would put the user back at step 3 for another test.
