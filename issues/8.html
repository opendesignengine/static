Feature #8

Add support for enhanced document management

Added by J. Simmons over 11 years ago. Updated about 11 years ago.

Status:Resolved Start date:04/08/2011
Priority:Normal Due date:
Assignee:- % Done:

100%
Category:-
Target version:v0.2 - Public Beta

Description

The idea is to allow users to have an easy and familiar way to control non-source code files. It should be versioned when needed and work similar to file management. Meta-data is a plus.

First implementation concept is using webdav module for easier sharing of engineering documents. Look at this plugin. Need to research what can be done to add version control support to this feature. Also look at integrating svn access through this connection.

Should also evaluate moving live ODE site from http to https before implementing this feature.
