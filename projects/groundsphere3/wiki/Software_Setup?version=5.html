Introduction¶

These instructions cover the software set up of the Ground Sphere Mk3.

Note, these steps are for Windows (tested on Windows 10).

1. Download required Software¶

  1. Download the following files
    1. "Windows SDR Package - SDR Sharp": https://airspy.com/download/ (ver 1.0.0.1443)
    2. "WXtoImg": https://wxtoimgrestored.xyz/downloads/ (ver 2.10.11)
    3. "VBCable": https://www.vb-audio.com/Cable/ (pack 43)

2. Install SDR Tools¶

  1. TODO
  2. Open SDR Sharp
  3. Select WFM in the Radio section
  4. Set the bandwidth to 40000 in the Radio section

3. Install WXtoImg¶

  1. TODO
  2. Open WXtoImg
  3. Modify your location
    1. Select Options -> Ground Station Location
    2. Modify latitude and longitude to match your ground station location
  4. Select File -> Update Keplers to update NOAA satellite TLEs

4. Install VBCable¶

  1. TODO
  2. Modify a setting in SDR Sharp
    1. Run SDR Sharp (if not already open)
    2. In the Audio section, select the Output selector to CABLE Input
