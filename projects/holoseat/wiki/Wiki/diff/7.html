Welcome to the Holoseat Wiki. Holoseat

The *Holoseat* is a gaming peripheral for PCs (it works with Windows, Mac OS, and Linux) that an open source Arduino project which turns your video games activity on exercise bikes/elliptical machines (generally recumbent models) into workouts. Every step you take in the game is driven keyboard command for locomotion (usually walking triggered by pedaling in the real world. *w* key). Using Holoseat is an immersive experience (players pedal faster when their characters are in danger) The current version only works with PCs and invisible. This combination masks the exercise experience in game play, transforming exercise from Macs, as they can have multiple keyboards for input devices (including joysticks if a chore into a natural part of a gamer's day. keyboard mapping program like "JoyToKey":http://www-en.jtksoft.net/ is used). In short, Holoseat _exerfies your games_!
